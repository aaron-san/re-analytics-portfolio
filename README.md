# Real Estate Analytics Portfolio

A collection of end-to-end analytics projects focused on real estate investment research — built to demonstrate SQL, data visualization, and forecasting skills in a commercial context.

Each project follows the same structure: a real public dataset, a business question worth answering, and a deliverable that would be useful in an actual investment or underwriting context.

---

## Projects

| Project | Dataset | Tools | Business Question |
|---|---|---|---|
| [Multifamily Rent Intelligence](./zillow-multifamily/) | Zillow ZORI + ZHVI | SQL · Tableau · Python | Which metros are decelerating, and where does rent vs. own math favor investors? |

## Stack

- **SQL** — SQLite for data modeling and analytical queries
- **Tableau** — Dashboard design and geographic visualization
- **Python** — Forecasting (Holt-Winters exponential smoothing), data wrangling (pandas)
